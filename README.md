# lostArk_api
- 로스트아크 OpenApi 호출을 위한 게이트웨이 서버
- lostArk_api 요청이 들어오면 OpenApi에 rest요청을 보내고 데이터를 리턴한다

## Stack
- Sping boot 3
- Jpa
- QueryDsl
- Spring Security

refactor : 리펙토링
